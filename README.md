# Документация 

### Описание проекта
Курсовой проект представляет собой single-page-application на тему фитнесс трекера. Фитнесс трекер это сервис, который позволяет пользователя управлять наборами различных упражнений. Наборы упражнений (комплексы) помогают пользователю при тренировках. Пользователь запускает комплекс на выполнение, в течении всего компелкса проигрывается видео, с тем или иным упражнением, идет обратный отсчет до конца комплекса и до конца текущего упражнения. Также предусмотрен отдых между упражнениями и между подходами в упражнение( обычно упражнение состоит из 4-5 подходов по 10-15 раз). Пользователь может создавать свои комплексы упражнений из своих собственных упражнений или уже готовых. Изначально присутствует небольшой набор базовых комплексов и упражнений.

1. На этой странице расположен слайдер, которые крутится постоянно.

### Описание модального окна (Регистрация/Авторизация)
![](http://dl1.joxi.net/drive/0006/2683/399995/151223/6229ca0f29.png)
1. Кнопка (Log in Facebook) осуществит вход через Facebook
2. Кнопка (Log in VKontakte) в данной версси приложения не работает
3. В полях осуществляется валидация по мере того как пользоваетель вводит данные. (на поле email наложено ограничение, которое обязует пользователя ввести строку, содержащую @, других проверок не проиводиться, ибо не за чем).

В следующей версии приложения пользователю будет высылаться письмо на указанный им почтовый ящик. В связи с чем проводится проверка только на существование @ в адресе.

После того, как пользователь заполнит все поля и они окажуться валидными сниметься ограничение на кнопку (Sign in/Sign Up). Произойдет авторизация/регистрация соответственно.

### Страница Тренировка
![](http://dl3.joxi.net/drive/0006/2683/399995/151223/b3134d83dd.png)

Страница предназначена для отображения и создания комплекосв упржанений. Сначала идут базовые упражнение, после, пользовательские.

#### Создание нового комплекса

Для того, чтобы создать новый комплекс упражнений, необходимо кликнуть на кноку (Создать).
![](http://dl3.joxi.net/drive/0006/2683/399995/151223/798e87e5d0.png)

Необходимо ввести название Нового комплекса (стоит проверка на длину и обязательность заполнения этого поля).
С помощью кликов по "Крестикам" добавить необходимые упражнения в новый комплекс.
После того,как комплекс будет создан, он динамически добавиться в список со всеми комплексами упражнений.

>Также указывается из каких упражнений состоит комплекс и общее время, которое пользователь затратит, выполняя этот комплекс.

### Страница Упражнений

### Личный кабинет пользователя

![](http://dl1.joxi.net/drive/0006/2683/399995/151223/9284c76987.png)

Фотография пользователя берется из социальной сети. В будущих версиях она будет выбираться пользователем (из доступных аватарок или же ему предоставиться возможность самому загрузить картинку).

Электронный адрес почты менять нельзя. Но возможно изменить пароль. В последствии для тех пользователей, которые зарегестрировались в приложении через социальные сети, я добавлю возможность ввести email, для того, чтобы отправлять им уведомления и различные информативные сообщения.

Присутствует удобный и красивый календарик для выбора даты.

![](http://dl3.joxi.net/drive/0006/2683/399995/151223/050d60bc2e.png) 
> Невероятно удобный и красивый!

### Как будем выполнять упаржнения?!

Самое главное, как и где будут "воспроизводится" комплексы упражнений?

Попасть сюда можно, щелкнув но любой из комлексов на странице "Тренировка"

![](http://dl1.joxi.net/drive/0006/2683/399995/151223/0f65c57406.png)

1. Ведется отсчет подходов (сколько подходов до конца комплекса)
2. Время одного подхода (каждое упражнение может состоять из нескольких подходов)
3. Общее время до конца комплекса.
4. Название упражнения, которое выполняется в данный момент.

При запуске видео стартует "воспроизведение" всего комплекса. Если видео заканчивается раньше, чем закончилось упражнение, то видео повторяется снова и снова, пока не закончится упражнение.
